# calconv
Calendar Converter  
(↑そのまんまそれ)

## Summary
webサイト作成の見づらいカレンダーを.csv等のGoogleカレンダーなどで読み込める形式に変換します。  
各webサイトのhtmlにものすごく依存するため、他のサイトへの転用は難しいと思います。

Linux, Python3での動作確認、開発をしています。
